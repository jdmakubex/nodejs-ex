Unidad 4. Interactividad

Descrición de las actividades

Desarrollar una aplicación WEB móvil (trabajo en equipo), que haga uso de todas las etapas descritas anteriormente (4.1), dicha aplicación debe ser Responsiva (C#, Java, el lenguaje está abierto a su elección) y para su creación se requiere hacer uso de librerías como HTML 5, JavaScript, CSS, Bootstrap, Angular JS, etc.

Así mismo, dicha aplicación deberá hacer uso de cualquier API (puede ser la de Googleo QGIS) o librería (ustedes la seleccionarán) que permita visualizar un mapa geográfico y mediante el uso de alguna consulta generada a partir de herramientas de análisis (el usuario mediante la aplicación podrá seleccionar algunas variables para llevar a cabo su consulta), visualizar en un mapa geográfico información estadística que se obtendrá de una base de datos pública.

El proyecto deberá estar o hacer uso de un servicio en la nube (Ustedes deciden cuál de sus servicios estará en la nube: Aplicación, Base de datos, Software de análisis. etc.). El Profesor tendrá acceso a su Aplicación desde cualquier dispositivo móvil para la consulta de información previa selección de una consulta, así como acceso a la nube para verificar el servicio. Deberán entregar código fuente utilizado en su aplicación, así como código de algoritmos utilizados para el Análisis de información, archivos de datos, etc. en archivo ZIP.
